## Escalonamento de Processos
O escalonamento de processos é uma atividade organizacional feita pelo escalonadorda CPU ou de um sistema distribuído, possibilitando executar os processos mais viáveis e concorrentes, priorizando determinados tipos de processos, como os de I/O Bound e os CPU Bound. Esta aplicação abrange dos algoritmos de escalonamento por Prioridade e FIFO.
